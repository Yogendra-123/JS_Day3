    4.What is callback hell

    callback hell is a pattern with multiple nested call backs which makes code hard to read and debug the code when we are dealing with asynchrous logic.The callback hell looks like a pyramid structure.

                    function print(){
                    setTimeout(()=>{
                        console.log("yogendra")
                        setTimeout(()=>{
                            console.log("patidar")
                            setTimeout(()=>{
                                console.log("khargone")
                                setTimeout(()=>{
                                    console.log("indore")
                                },1000)
                            },2000)
                        },3000)
                    },4000)
                   }
                   print();
